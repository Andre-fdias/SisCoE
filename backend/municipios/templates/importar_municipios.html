Instruções para importação de dados de Postos

  • O arquivo deve estar no formato .CSV (separado por vírgula ou ponto e vírgula) ou .XLSX/.XLS (Excel).
  • A primeira linha do arquivo deve conter os seguintes cabeçalhos de coluna, exatamente como mostrado:
    • sgb
    • posto_secao
    • posto_atendimento
    • cidade_posto
    • tipo_cidade
    • op_adm
    • quartel (opcional)
    • data_criacao (opcional)
    • usuario_id (ID de um usuário existente)
    • telefone
    • rua
    • numero
    • complemento (opcional)
    • bairro
    • cidade_contato
    • cep
    • email
    • longitude_contato
    • latitude_contato
    • cel
    • ten_cel
    • maj
    • cap
    • tenqo
    • tenqa
    • asp
    • st_sgt
    • cb_sd
    • descricao_cidade (opcional)
    • municipio
    • longitude_cidade
    • latitude_cidade
    • bandeira (opcional)
  • Certifique-se de que os valores nas colunas correspondam aos tipos esperados (texto, número inteiro, decimal, data).
  • Para os campos com opções (`choices`), os valores no arquivo devem corresponder exatamente às chaves definidas no seu modelo.
  • O sistema tentará converter datas em diversos formatos comuns.
  • Certifique-se de que o usuario_id corresponda a um usuário existente no sistema.
  • O tamanho máximo do arquivo permitido é de 10MB.
  • Para referência, você pode baixar um modelo de arquivo CSV.
